# Contributing

Thanks for taking the time to contribute.

## Development
1. Use the Node version in `.nvmrc` (for example, `nvm use`).
2. Install dependencies with `npm install`.
3. Start the dev server with `npm run dev`.

## Pull Requests
- Keep changes focused and describe the intent clearly.
- Run `npm run build` before submitting.
- For UI changes, include before/after screenshots or a short GIF.
- Update or add documentation when it helps reviewers.

## Code Style
- Follow existing component and styling patterns.
- Prefer small, readable components over large files.
- Avoid adding dependencies unless they are clearly justified.
